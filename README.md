# MUHIUM & ZAMIL VU Hub

Welcome to the most elegant and powerful student review hub inspired by **lgwat** and **paradoxical**. This project is a premium-style paper bank and exam review portal built for Virtual University students, designed to feel modern, polished, and beautifully easy to use.

## ✨ What This Project Is

`MUHIUM & ZAMIL VU Hub` is a frontend experience for sharing and discovering exam papers, midterm reviews, and course updates. It includes:

- A sleek landing interface with theme switching
- Course search for quick paper lookup
- Review submission forms for students to add exam experience
- Copy and WhatsApp sharing utilities for fast collaboration
- A refined visual design inspired by premium UI trends

## 🎯 Why It Stands Out

- Elegant layout with dark, desert, and light modes
- Responsive design for phone, tablet, and desktop
- Smooth interactive tab navigation and paper cards
- Smart clipboard sharing for review distribution
- UX built around fast student help and community sharing

## 💡 Built From

- `index.html` - main interface and layout
- `spring2026midsdata.js` - sample midterm review data
- `anti-inspect.js` - extra page protection logic
- `restricted.html` - secondary restricted access design

## 🚀 How to Use

1. Open `rvw/index.html` in your browser.
2. Search your course code (e.g. `CS101`, `MTH101`).
3. Read and share paper reviews with WhatsApp or clipboard.
4. Submit your own review using the form.
